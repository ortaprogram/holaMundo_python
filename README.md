# holaMundo_python
Página “Hola mundo” de iniciación con Python para el curso de Programación de Inteligencia Artificial (PIA01). 
Imprime en la consola el texto “Hola mundo”, ejecutando el archivo desde un terminal (desde Visual Studio Code, pulsar Ctrl+Ñ, ir a la carpeta contenedora del archivo y escribir python holaMundo.py). 
En este caso, es importante tener instaladas las extensiones de Python.
