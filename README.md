# holaMundo_python

## Página de Iniciación con Python

---

Página “Hola mundo” de iniciación con Python para el curso de **Programación de Inteligencia Artificial (PIA01)**.

### **Instrucciones de Ejecución**

* Imprime en la consola el texto **“Hola mundo”**.
* Para ejecutar el archivo desde un terminal (usando Visual Studio Code), pulsa `Ctrl + Ñ`.
* Navega a la carpeta que contiene el archivo.
* Escribe `python holaMundo.py` y pulsa Enter.

**Nota Importante:** Es fundamental tener instaladas las extensiones de Python en tu entorno de desarrollo.
